# crypto (rweather/Crypto) Component

This is the [rweather/Crypto](https://github.com/rweather/arduinolibs/tree/master/libraries/Crypto) library, ported for ESP-IDF.

**Status**: Built but currently unused. The FIPS transport layer uses mbedtls + micro-ecc instead.

**Reserved for**: Potential future use in Mesh V2 (additional crypto primitives) or as a fallback if mbedtls becomes a bottleneck on ESP32-C3.

**DO NOT REMOVE** — kept per project decision (see AGENTS.md).
