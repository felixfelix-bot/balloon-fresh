## Chip Overview

- **Semtech LR2021** = Gen 4 "LoRa Plus™" multi-band transceiver
- Backward compatible with previous LoRa generations (SX127x, SX126x, LR11x0)
- Multi-PHY: LoRa CSS, LR-FHSS, FLRC, FSK

## Frequency Bands

- Sub-GHz terrestrial: 150–960 MHz (standard 470/868/915 MHz)
- 2.4 GHz ISM band (global)
- S-Band (satellite/NTN)

## FLRC Mode (Fast Long-Range Communication)

- **Modulation:** Coherent Gaussian Filtered Minimum Shift Keying (GMSK)
- **Raw data rates:** 260 kbps to 2.6 Mbps
- **Bandwidth:** Variable, up to 2.06 MHz
- **FEC coding rates:** 1/2, 2/3, 3/4 (programmable)
- Unlike SX1280 (FLRC only at 2.4 GHz), LR2021 supports FLRC in Sub-GHz AND 2.4 GHz
- Solves LoRa's low throughput pain point

## SPI Command Architecture

- 4-wire SPI with dedicated BUSY, IRQ, NSS (CS) lines
- Host-controlled register-and-command driven
- Standard op-code state machine:
  1. **SetPacketType** — must be called before modulation-specific commands. Set PacketType = FLRC for high-speed mode
  2. **SetModulationParams** — configures Bit Rate/Bandwidth pair, Modulation Index, Pulse Shape (Gaussian filter)
  3. **SetPacketParams** — sync word length, preamble length, header type (variable/fixed), payload size, CRC config, whitening
  4. **Buffer Operations** — WriteBuffer/ReadBuffer with internal memory offset pointer

## Architecture Notes

- Gen 4 chip — NOT LR10xx series. Different register map, different command set.
- The SPI interface uses BUSY pin (HIGH during command processing + RF activity) and IRQ pin (interrupt on enabled events).
- DIO mapping configures which IRQ events trigger the IRQ pin.
- Multiple IRQ bits can be enabled simultaneously on the same DIO pin.
