# LR2021 Research Findings — 2026-07-16

## Chip: Semtech LR2021 Gen 4 "LoRa Plus"
- NOT SX1280, NOT LR10xx. Gen 4 multi-band transceiver.
- Multi-PHY: LoRa CSS, FLRC, GFSK, LR-FHSS
- FLRC = Fast Long-Range Communication using coherent GMSK

## Key Specs
- FLRC data rates: 260 kbps to 2.6 Mbps (max, confirmed)
- Frequency bands: Sub-GHz (150-960MHz), 2.4 GHz ISM, S-Band (satellite/NTN)
- FLRC available in BOTH Sub-GHz AND 2.4 GHz (unlike SX1280 which was 2.4GHz only)
- FEC: programmable coding rates 1/2, 2/3, 3/4
- Bandwidth: up to 2.06 MHz
- SPI: 4-wire SPI with BUSY, IRQ, NSS lines
- Backward compatible with previous Gen transceivers

## SPI Command Architecture
- Host-controlled register+command driven
- SetPacketType (opcode) → must run before modulation-specific calls
- SetModulationParams → bitrate/bandwidth, modulation index, pulse shape
- SetPacketParams → sync word, preamble, header type, payload size, CRC, whitening
- Buffer operations: WriteBuffer/ReadBuffer with internal offset pointer

## FLRC Mode Details
- Uses coherent Gaussian Filtered Minimum Shift Keying (GMSK)
- Resolves traditional LoRa throughput limits
- Coding rates 1/2, 2/3, 3/4 for robustness vs throughput tradeoff
- Our firmware uses: 2600 kbps, 8-symbol preamble, 4-byte sync, no CRC (uncoded)
