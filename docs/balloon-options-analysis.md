## Gas Type: The #1 Success Factor

### Combined Flight Data from All Sources

| Gas Type | Source | Flights | Best Result | Circumnavigations |
|----------|--------|---------|-------------|-------------------|
| **Hydrogen** | Ruthroff (37 flights) | ~9 | 507 days, 27 laps | 5/9 = **55%** |
| **Hydrogen** | Klofas/SF-HAB | 5+ | ~12 days, partial lap | Some partial |
| **Ultra-pure He (99.999%)** | Ruthroff | 3 | **528 days, 32 laps** | 2/3 = **67%** |
| **Ultra-pure He (99.999%)** | Ruthroff | 1 (mixed w/ party) | 5 days 15h | 0/1 |
| **Party-store He (80/20)** | Ruthroff | 9 | 9 days | **0/9 = 0%** |
| **Party Factory He (Germany)** | KI4MCW (31 flights) | ~15 | 15 days, nearly circumnavigated | 0 full |
| **Supermarket He** | IEEE Spectrum (Schneider) | 3 | Crossed to Mediterranean | 0/3 |
| **Party-store He (mixed brands)** | Ruthroff purity test | N/A | Hover test only | Spoozer: 4.15g, Party City: 7.25g, Ultra-pure: 8.42g |

### Key Finding

**Party-store helium is universally inadequate for long-duration flights.** Ruthroff's data is the most conclusive: 0% circumnavigation rate with party-store helium across 9 flights, vs 55% with hydrogen and 67% with ultra-pure helium.

### Helium Purity Lift Comparison (Ruthroff's Test)

Ruthroff conducted a controlled experiment: inflate identical Yokohama balloons with different gases, hang weights until hover.

| Gas Source | Hover Weight | Relative to Ultra-Pure |
|-----------|-------------|----------------------|
| Spoozer (party) | 4.15g | 49% |
| Party City (party) | 7.25g | 86% |
| 99.999% industrial He | 8.42g | 100% |

**Note:** In the US, balloon helium must legally contain ≥20% oxygen (to prevent asphyxiation from inhalation). This effectively caps party-store helium at ~80% purity.

### Recommendation

**Use hydrogen for all long-duration flights.** It is cheaper, provides ~8% more lift than helium, and has a proven track record. For safety-constrained situations, use industrial-grade 99.999% helium from a gas supplier (NOT party-store).

## Balloon-by-Balloon Analysis

### Yokohama Balloon Co. (Japan) — RECOMMENDED

**The sweet spot for long-duration amateur pico-balloon flights.**

| Parameter | Value |
|-----------|-------|
| Manufacturer | Yokohama Balloon Co., Yokohama, Japan |
| Original purpose | Wedding/party decoration |
| Diameter (inflated) | ~90cm (~3 feet), just under 100" circumference |
| Weight | 47g (confirmed by Ruthroff) |
| Cost | $150 for 10-pack including shipping = **$15 each** |
| Nozzle types | Self-sealing (older) and no-nozzle (newer) |
| Proven altitude | 38,000-51,000 feet (11.6-15.5 km) |
| Proven duration | 528 days, 32 circumnavigations (JR29, Ruthroff) |
| Material | Trade-secret foil (not standard Mylar) |
| Film thickness | Estimated 1-2 mil (25-50 µm) |

**Preparation procedure (from Ruthroff):**
1. Inflate with air pump to 100-116" circumference (stretch)
2. Hold at stretch for hours to days
3. Deflate completely (2 hours with pump)
4. Refill with lifting gas (~0.07 m³ for launch)
5. Heat-seal nozzle (setting "6" on standard heat sealer, ~5 seconds)
6. Add Kapton/Kynar tape over seal
7. Measure neck lift with calibrated weights
8. Launch with 5-8g free lift

**At launch, Yokohama looks underinflated** — this is normal. The balloon expands as it rises to altitude.

**Where to buy:**
- US resellers ship 10-packs for $150
- Contact Yokohama Balloon Co. directly for international orders
- Community members sometimes share orders via picoballoon mailing list

### Chinese Clear Party Balloons (36") — Budget Option

**Used by SF-HAB group (Klofas et al.) for many flights.**

| Parameter | Value |
|-----------|-------|
| Source | AliExpress, various sellers |
| Cost | ~$1-2 each |
| Size | ~90cm (36") when inflated |
| Standard config | 2 balloons per flight |
| Gas | Hydrogen |
| Best result | ~12 days, partial circumnavigation (Launch 19 to Midway Islands) |
| Typical altitude | 32,500-36,500 ft (9.9-11.1 km) |

**SF-HAB flight records (from Klofas blog):**
- Launch 12-16 (Oct 2021): 5 launches in 2 weeks, only 1 survived first night
- Launch 18 (Nov 2021): Almost circumnavigated, lost over Uzbekistan
- Launch 19 (Dec 2021): Reached Midway Islands in 12 days

**Assessment:** Workable for short flights with hydrogen, but reliability is low. Seam quality varies between batches. Not recommended for mesh network where consistent multi-week flights are needed.

### PartyWoo 50" Silver Balloons — Intermediate

**Tested extensively by KI4MCW (Bryan, 31 flights).**

| Parameter | Value |
|-----------|-------|
| Cost | ~$5 each |
| Size | ~127cm (50") |
| Best result | 15 days, nearly circumnavigated (Flight #14) |
| Typical result | 4-10 days, often Atlantic splashdown |

**KI4MCW's experience:**
- Flight #14 (Feb 2025): Pre-stretched PartyWoo 50" + heat-sealed neck + German helium = 15 days, fell short by ~2000 miles
- Flight #15-16: Both ended in Atlantic within 5 hours
- Flight #17: 4 days 21h, burst over Siberia
- Flight #18: 6 days 23h, disappeared over Atlantic
- Flight #24: Sprang a leak, 6.5 hours
- "I'm frustrated by the continued failures with the PartyWoo balloons"

**Assessment:** Better than small 18" DecoGlee, worse than Yokohama. Heat-sealing helps but consistent leaks remain a problem. Not recommended for primary flights.

### Cymylar 60" Clear Balloons — Unproven

**Also tested by KI4MCW as a Yokohama alternative.**

| Parameter | Value |
|-----------|-------|
| Cost | ~$5-10 each |
| Size | ~150cm (60") |
| Best result | 7 days 18h (Flight #25) |
| Typical result | 2-7 days |

**KI4MCW's experience:**
- Flight #25: Sealed both top and bottom with Kapton, topped off helium twice pre-launch = 7 days 18h, lost to thunderstorm
- Flight #26: Lost lift overnight, never transmitted
- Flight #27: 8 hours, developed leak
- Flight #28: 4 days 2 hours, burst near Turkey

**Assessment:** Too inconsistent. The larger size provides more lift but leak problems persist.

### DecoGlee 18" Foil Balloons — Development Only

**Our tested balloons. Adequate for bench testing, inadequate for flight.**

See `docs/balloon-test-results.md` for full test data.

| Parameter | Value |
|-----------|-------|
| Cost | EUR 0.37 each (30-pack) |
| Net lift | ~4.8g per balloon |
| Envelope weight | ~10.5g per balloon |
| Indoor leak rate | 0.15 g/day per balloon |
| Estimated outdoor duration | 4-8 days |

**Dead balloon penalty:** 10.5g dead weight vs 4.8g lift — a dead balloon that stays attached is 2.2x heavier than the lift it provided. Cut-down is essential for multi-balloon rigs.

**Assessment:** Only useful for indoor bench testing and very short first test flights. Not viable for mesh network.
