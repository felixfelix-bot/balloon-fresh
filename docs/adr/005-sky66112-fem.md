# ADR 005: SKY66112-11 als Front-End-Module (PA + LNA)

## Status
Akzeptiert

## Kontext
Der LR2021 liefert intern maximal +12 dBm bei 2.4 GHz. Fuer Reichweiten von 100+ km zur Bodenstation ist ein externer Leistungsverstaerker (PA) fuer TX und ein Vorverstaerker (LNA) fuer RX sinnvoll.

## Entscheidung
**Skyworks SKY66112-11** als 2.4 GHz Front-End-Module (FEM) mit integriertem PA, LNA und SPDT Antennen-Schalter.

## Spezifikationen

| Parameter | Wert |
|-----------|------|
| Frequenz | 2.4 GHz ISM |
| PA Ausgangsleistung | **+22 dBm** (158 mW) |
| PA Strom (@ +22 dBm) | ~90 mA |
| LNA Gewinn | **+14 dB** |
| LNA Noise Figure | **1.8 dB** |
| SPDT Insertion Loss | ~0.5 dB |
| Package | 16-pin QFN, 3x3 mm |
| Gewicht | ~0.1g |
| TX/RX Umschaltung | GPIO-gesteuert (CPS, CTX Pins) |

## Alternativen

| Option | TX Power | LNA Gain | Gewicht | Bemerkung |
|--------|----------|----------|---------|-----------|
| **SKY66112-11** | +22 dBm | +14 dB | 0.1g | Best Choice |
| Qorvo RFFM4227 | +20 dBm | +13 dB | ~0.2g | Alternative |
| NiceRF LoRa1280F27 | +27 dBm | integriert | ~5g (Modul) | Nur SX1280 |
| Kein FEM | +12 dBm | - | 0g | Minimal-Reichweite |

## Begruendung

1. **PA + LNA + Switch in einem IC**: Spart Gewicht und Board-Flaeche
2. **+22 dBm TX**: Boostet LR2021's +12 dBm auf +22 dBm = +10 dB mehr Link-Budget
3. **+14 dB LNA**: Verbessert RX-Sensitivity drastisch fuer Bodenstations-Kommandos
4. **Extrem leicht**: 0.1g, QFN 3x3mm Package
5. **2.4 GHz optimiert**: Speziell fuer LoRa/BLE/Zigbee im ISM-Band
6. **GPIO-steuerbar**: Einfache TX/RX Umschaltung durch ESP32-C3

## Power-Budget Auswirkung
- TX Burst bei +22 dBm: ~90 mA fuer ~0.5s (LoRa SF10) = 45 mAs
- Mit 1.65F Supercap-Bank: Spannungsabfall = I*t/C = 0.09*0.5/1.65 = 0.027V
- Akzeptabel - Supercaps halten problemlos durch
